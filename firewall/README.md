# firewall.sh

#### Simples script de firewall

### Opcoes:
- --start	Inicia firewall 
- --stop	Para firewall
- --restart	Reinicia o firewall
- --help	Mostra a Ajuda

### Exemplo de uso: 
sudo sh firewall.sh --start
sudo sh firewall.sh --stop
